
## Node Version Manager

`.nvmrc` - node version manager runtime configuration.

```sh
touch .nvmrc
```

List nvm Vresion
```sh
nvm ls
```

Nvm use - switch to that version which uses in current project
```sh
nvm use
```
if node of specified in `.nvmrc` version is not installed then use 
```sh
nvm install
```
